Email Application:

Scenario: YOu are an IT Support Adminstrator Specialist and are charged with the task of 
creating email accounts for new hires.

Your application should do the following:

* Generate an email wiht the following syntax: firstname.lastname@department.company.com
* Determin the department ( sales,development,accounting), if non leave blank
* Generate a random String for a password
* Have set methods to change the password, set the mailbox capacity, and define an alternate email address
* Have get methods to display the name, email, and mailbox capacity.
